L'objectif de ce document est de proposer un guide de démarrage pour la collaboration au projet Polytech Abroad

Présentation

1. Objectifs du site

L’entraide

  • à l’étranger
  • à la préparation d’un départ

La création de liens

  • professionnels (recherche d’emploi et de stage)
  • sociaux

La mise en avant de l’école

  • pour les entreprises recruteuses
  • pour les futurs étudiants

2. Contexte du projet

Site géré par des étudiants de Polytech pour des étudiants de Polytech

Ouvert à la collaboration

2. Contribuer au projet

Pour contribuer au projet du développement du site Polytech Abroad :

1. Apprendre le fonctionnement des frameworks utilisés

  • back office : CakePHP - http://cakephp.org
  • front office : Bootstrap (twitter) - http://getbootstrap.com

2. Télécharger le répertoire sur Github : https://github.com/richfab/POLY_INT

3. A vous de jouer !
