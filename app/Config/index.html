Regatime

html css javascript ajax mysql php

Early 2013, as I was attending a rowing contest, I realized that spectators could be more entertained if the results were being displayed in real time on a screen and online. So I decided to study the real needs and to work with the president of the rowing club to release a first prototype of the online timer which was used last February during a contest. It was a true success : the rowers, the spectators and the organizers were so happy with the change it had brought to the contest that I decided to take the project to the next step. Since summer 2013, I have been working on making the website accessible by any rowing club and thinking of a possible business plan.

Roadtrip

html css javascript mysql php

As I was traveling across the USA, I wanted to find a way to share my adventure with my family and friends. I found that an interactive map was the best way for them to see what progress I had made and how many miles were left ahead of me. This project taught me how to focus on the important parts because of the very short time I had to develop the website (a few days). The interface had to be simple and my position had to be updatable via my smartphone.
